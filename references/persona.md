# 博主人设包（人设即资产）

把账号识别度沉淀成可复用配置：一次录入，张张复用；多账号运营只需切换人设包。

## 配置文件

人设包存为 JSON（样例见 `assets/persona-sample.json`）：

```json
{
  "id": "moyu-xiaoli",
  "nickname": "摸鱼小李",
  "tagline": "每天一个 AI 摸鱼技巧",
  "avatar_emoji": "🐟",
  "catchphrase": "关注我，下班早一小时",
  "end_checklist_title": "今天的重点",
  "hashtags": ["#AI工具", "#效率提升", "#摸鱼技巧"]
}
```

## 注入点（每张卡自动带）

| 位置 | 内容 | 规则 |
| --- | --- | --- |
| 每卡右下角 | `avatar_emoji + nickname` 角标 | 字号 28–32px，灰阶色，不抢正文 |
| 封面底部 | `tagline` | 可选，一行 |
| 结尾页 | `end_checklist_title` + 总结清单 + `catchphrase` | 固定模板，不现场发挥 |
| 发布文案 | `hashtags` 拼入话题标签 | 3–6 个 |

## 规则

- 人设元素只用灰阶或主题主色，不引入新颜色。
- 未提供人设包时：只问一次（昵称 + 一句定位），其余字段自动补全；用户拒绝则省略人设元素。
- 同一人设包在所有主题下通用（主题只换皮，人设不变）。
