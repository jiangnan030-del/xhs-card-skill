# 主题生成器：描述 / 参考图 → 新主题

内置 6 套不够用时，现造一套。流程与 gzh-design 同构，产物适配卡片三段式。

## 流程（4 步）

1. **收集偏好（一次问全，不逐条追问）**
   - 必填：主题描述（一句气质描述）或参考图。
   - 可留空自动补全：名称 / 主色 / 背景色 / 正文色 / 荧光标记色 / 圆角 / 阴影 / 适用题材。
2. **生成区块库**：产出完整预览 HTML（封面模板 ×3 + 内容页组件 ×8–12 + 结尾页 ×2），存 `assets/theme-previews/{id}.html`，浏览器整页确认风格（不逐块问）。
3. **转标准主题库 + 登记**：确认后转成 `references/theme-{id}.md`，补齐五章节（变量表 / 封面模板 / 内容页组件 / 结尾页模板 / 配方表），登记进 `theme-index.md`，跑 `scripts/card_lint.py` 到 0 ERROR。
4. **即刻同权**：之后与内置主题完全一样，直接说「用 XX 主题排这篇」。

## 配色角色分工（自动推导整套色板）

| 角色 | 作用 | 取色思路 |
| --- | --- | --- |
| 主色 | 封面大字/序号/锚点（每卡 ≤3 处） | 有辨识度的品牌色 |
| 浅底/浅边框 | 要点卡背景、标签底 | 主色 + 大量白 |
| 荧光标记色 | 每卡 1–2 处关键句背景 | 主色浅色版或冷暖对比色 |
| 中性灰阶 | 正文/辅助/分割（承担 90% 文字） | `#111827 → #9CA3AF` |
| 人设角标色 | 右下角昵称 | 灰阶中段 |

**可读性底线**：正文与背景对比度 ≥ 4.5:1；深底主题（如霓虹磁带）正文用 `#E9ECEF` 以上亮度。
